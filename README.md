> 방문자 증가 봇

## Installation

```sh
pip install -r requirements.txt
```



## Usage example

```python
if __name__ == "__main__":
    url = "<URL>" # Target URL
```

url 변수에 조회수를 증가시 킬 대상의 URL 주소를 입력합니다.
